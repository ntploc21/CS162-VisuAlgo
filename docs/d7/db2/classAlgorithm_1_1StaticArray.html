Detailed Description

The algorithm class that is used to generate step-by-step instructions for the visualization of the static array.

Note
In this Static Array algorithm class, the visualizer I use for the static array is a dynamic array with a fixed size (and hide all the inaccesible elements). This is because the visualzation of the static array and the dynamic array have a lot in common.
